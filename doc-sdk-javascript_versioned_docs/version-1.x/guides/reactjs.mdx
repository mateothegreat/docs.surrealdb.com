# React.js

React is a popular JavaScript library for building user interfaces. In SurrealDB, using the native JavaScript SDK, within TanStack Query, with support for data caching and syncing, and authentication you can build a React application that can query a remote database instance.

In this guide, you will learn how to integrate SurrealDB with a React application and perform basic data queries. The guide will cover the following topics:

- Setup a React application with Vite.
- Importing sample data into SurrealDB.
- Using SurrealDB methods to perform data queries.
- Displaying data in a React application.
